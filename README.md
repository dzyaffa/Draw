# 🎨 Draw Learning Roadmap

A structured roadmap website for learning how to draw, from basic fundamentals to coloring and rendering.

🌐 Live Demo:
https://dzyaffa.github.io/Draw/

---

## 📖 About

Draw Learning Roadmap is a beginner-friendly guide designed to help aspiring artists learn drawing step by step.

Instead of randomly watching tutorials, learners can follow a clear progression path:

1. Lines & Shapes
2. Head Construction
3. Eyes
4. Hair
5. Body & Anatomy
6. Hands
7. Coloring & Rendering

The goal is to provide a clear learning order and curated resources for each phase.
